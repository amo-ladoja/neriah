# PWA Icons

Place your PWA icon files here with the following sizes:

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

All icons should be square and in PNG format.

You can use a tool like https://realfavicongenerator.net/ to generate all required sizes from a single source image.
